// USAGE INSTRUCTIONS: English

// - This file defines the different game modes available on the server.
// - Each game mode is organized under a category (for example, "Friendly Fire").
// - Inside each category, there are specific configurations for each mode, with their name and path.
//
// HOW TO EDIT:
// - To remove a mode, simply delete its entry from the file.
// - To add a new mode, follow this format:
//   "CategoryName"
//   {
//       "Path/To/Mode"       // That is, the .cfg file inside cfg/
//       {
//           "name" "Visible name in the voting menu"
//       }
//   }
//
// CUSTOMIZATION EXAMPLE:
// If you want to add a mode called "Friendly Fire 10", with its configuration in "cfg/friendlyfire/friendlyfire10.cfg",
// add it inside the corresponding category:
//
// "Friendly Fire"
// {
//     "friendlyfire/friendlyfire10"
//     {
//         "name" "Friendly Fire 10"
//     }
// }
//
// NOTE:
// - CVAR changes are automatically saved between map changes if you use sm_cvar inside the .cfg files.
// - No server restart is needed to apply modes.
// - You can create multiple variations such as:
//     friendlyfire10.cfg
//     friendlyfire20.cfg
//     friendlyfireoff.cfg
//     friendlyfireinverse.cfg
// - New categories can be added if needed.
